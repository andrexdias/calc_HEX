## Instalação e inicialização

É preciso ter o python3 para conseguir rodar o projeto.

```bash
  https://www.python.org/downloads/
```
    
## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/andrexdias/calc_HEX
```

Entre no diretório do projeto

```bash
  cd calc_HEX
```

Inicie o arquivo

```bash
  python3 main.py
```
